<h1>GoBarber</h1>
<p>
Aplicação para o gerenciamento de estabelecimentos de estética.</br>
Projeto desenvolvido durante o Bootcamp da Rocketseat, estudando NodeJS, ReactJs e React Native.
</p>

<h2>Ferramentas</h2>
<ul>
  <li>Express</li>
  <li>Node</li>
  <li>Yarn</li>
</ul>

<h2>Inicialização</h2>
<p>
Com yarn e node instalados na máquina:
</p>

```js
  $ node src/server.js
```
