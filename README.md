<h1>GoBarber</h1>
<p>
Aplicação para o gerenciamento de estabelecimentos de estética.</br>
Projeto desenvolvido durante o Bootcamp da Rocketseat, estudando NodeJS, ReactJs e React Native.
</p>

<h2>Ferramentas</h2>
<ul>
  <li>Bcryptjs</li>
  <li>Date-fns</li>
  <li>Docker</li>
  <li>EditorConfig</li>
  <li>Eslint</li>
  <li>Express</li>
  <li>JWT -JsonWebtoken</li>
  <li>Multer</li>
  <li>Node</li>
  <li>Nodemon</li>
  <li>PostBird</li>
  <li>Postgres</li>
  <li>Prettier</li>
  <li>Sequelize</li>
  <li>Sucrase</li>
  <li>Yarn</li>
  <li>Yup</li>
</ul>

<h2>Inicialização</h2>
<p>
Com yarn e node instalados na máquina:
</p>

```js
  $ yarn #Instala as dependências 
  $ node src/server.js #Inicia o servidor
```
<p>Rodar o Docker:</p>

```js
  $ sudo docker start database
  $ sudo docker start mongobarber
```
